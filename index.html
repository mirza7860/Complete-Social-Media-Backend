Your Web App Routes and Functionality

BaseURL: http://localhost:4000

Route Middleware Data from Request Response
POST /auth/register Middleware: None Data from Request: User registration details Response: User registration status
POST /auth/login Middleware: None Data from Request: User login credentials Response: Authentication status
POST /posts Middleware: Authentication required Data from Request: New post details Response: Post creation status and feed posts
POST /posts/:userId/posts Middleware: Authentication required Data from Request: userId Response: Single User Posts
POST /posts/:userId/posts Middleware: Authentication required Data from Request: ViewersId Response: Likes
GET /user/:id Middleware: Authentication required Data from Request: User ID Response: User information
GET /user/:id/friends Middleware: Authentication required Data from Request: User ID Response: List of user's friends
PATCH /user/:id/:friend Middleware: Authentication required Data from Request: User ID, Friend ID Response: Updated user's friend list
